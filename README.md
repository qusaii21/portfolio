# Qusai Shergardwala — Portfolio

A modern, interactive 3D developer portfolio built with React, TypeScript, Three.js, and GSAP.

## Tech Stack

- React + TypeScript
- Three.js / WebGL
- GSAP animations
- Vite

## Getting Started

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```
